Тестовое задание на Front-end разработчика Onpoint 

Демо: https://onpoint-test-two.vercel.app/

Использованные технологии:
 - React
 - Webpack
 - Prettier
 - Babel
 - BEM
 - PostCSS
 - JS

Режим разработчика:
```
npm run dev
```

Сбилдить:
```
npm run build
```

Запустить билд на Express:
```
npm run start
```
